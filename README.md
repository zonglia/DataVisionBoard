# 1. 技术选型

# 1.1 核心框架

- ​​Vue 3​​：使用最新的 Vue 版本，提供更好的性能和组合式 API
- ​​TypeScript​​：强类型支持，提高代码健壮性
- ​​Pinia​​：轻量级状态管理
- ​​Vite​​：下一代前端构建工具，极速开发体验

# 1.2 可视化组件

- ​​ECharts：专业级图表库，支持丰富可视化效果
- ​​DataV Vue3​​：大屏专用组件库（含滚动表格等特效组件）

# 1.3Vite 插件 ​​：

- @vitejs/plugin-vue：Vue 单文件组件支持
- unplugin-auto-import：自动导入 API，减少样板代码
- unplugin-vue-components：自动导入 Vue 组件
- vite-plugin-svg-icons：SVG 图标处理
